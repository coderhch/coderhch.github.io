http_random_index_module模块

该ngx_http_random_index_module模块处理以斜杠字符（’ /‘）结尾的请求，并在目录中选择一个随机文件作为索引文件

配置示例

  1. 装备好3和html文件

1.html=>红色，2.html=>绿色，3.html=蓝色

  1. 编辑default.conf文件vim /etc/nginx/conf.d/default.conf

location / {
root /opt/app/code/;
random_index on;
}

  1. 检查语法是否正确nginx -tc /etc/nginx/nginx.conf

nginx: the configuration file /etc/nginx/nginx.conf syntax is ok
nginx: configuration file /etc/nginx/nginx.conf test is successful

  1. 启动nginx服务nginx -c /etc/nginx/nginx.conf
  2. 访问http://ip.com时就从配置的页面中随机挑选一个页面显示

指令

Syntax: random_index on | off;
Default: random_index off;
Context: location
