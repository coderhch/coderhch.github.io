# pwd和which命令

## pwd

### 描述

pwd是Print Working Directory（打印当前工作目录）的缩写

### 用法

```shell
[root@iZm5e9aiazlx2ktkg01676Z home]# pwd
/home
```

## which

### 描述

获取命令的可执行文件位置

### 用法

```shell
[root@iZm5e9aiazlx2ktkg01676Z home]# which pwd
/usr/bin/pwd
```
